# SkynetAccessibility Scanner

## Overview

SkynetAccessibility Scanner is a powerful Plone accessibility testing and scanning module designed to help organizations identify, monitor, and fix accessibility issues across their websites. It enables businesses, agencies, and government organizations to maintain compliance with global accessibility standards while improving user experience for all visitors.

Built for Plone 6, this module provides automated scanning, detailed reporting, and continuous monitoring - making accessibility management simple and scalable.

## Installation

### Prerequisites

Before installing SkynetAccessibility Scanner, make sure your setup meets these requirements:

- **Plone version:** 6.1.2
- **Python version:** 3.11

> If Plone is not yet installed, follow the [official Plone installation guide](https://6.docs.plone.org/admin-guide/install-pip.html).

---

### Option A: Install via pip (Recommended)

**Step 1** — Open your terminal and navigate to your Plone project folder:

```bash
cd /path/to/your/plone/project
```

> Replace `/path/to/your/plone/project` with the actual folder path of your Plone site.

**Step 2** — Install the plugin using pip:

```bash
pip install plone.skynetaccessibility_Scanner
```

Wait for the installation to complete. You will see a success message once it's done.

**Step 3** — Restart your Plone site:

```bash
cd mysite
runwsgi etc/zope.ini
```

**Step 4** — Activate the plugin in Plone:

1. Log in to your Plone site as an administrator.
2. Go to **Site Setup** → **Add-ons**.
3. Find **SkynetAccessibility Scanner** in the list.
4. Click **Install**.

The plugin is now active on your site!

---

### Option B: Install from a Downloaded ZIP File

Use this method if you have downloaded the plugin manually.

**Step 1** — Download and extract the ZIP file into your Plone project directory.

**Step 2** — Install the extracted package via pip:

```bash
pip install -e /path/to/plone_skynetaccessibility_Scanner
```

> Replace `/path/to/plone_skynetaccessibility_Scanner` with the folder where you extracted the ZIP file.

**Step 3** — Restart your Plone site:

```bash
cd mysite
runwsgi etc/zope.ini
```

**Step 4** — Activate the plugin in Plone:

1. Log in to your Plone site as an administrator.
2. Go to **Site Setup** → **Add-ons**.
3. Find **SkynetAccessibility Scanner** in the list.
4. Click **Install**.

The plugin is now active on your site!

---

### Deactivating the Plugin

If you need to turn off the plugin at any time:

1. Go to **Site Setup** → **Add-ons**.
2. Find **SkynetAccessibility Scanner** in the list.
3. Click **Uninstall**.

## CORS Policy Configuration

The plugin communicates with an external server to function properly. Ask your server administrator to whitelist the following URL in your server's CORS (Cross-Origin Resource Sharing) configuration:

| Domain | Description | Usage |
|--------|-------------|-------|
| `https://skynetaccessibilityscan.com` | Skynet Accessibility Scan (Global Domain) | API access and resources |

### Instructions

1. Update your server's CORS configuration to include the URL above.
2. Ensure wildcard subdomains (`*`) are supported where necessary.
3. Verify application functionality by testing requests to this domain.
4. If issues persist, consult the documentation for CORS configuration guidance.
